# Xilerast's OpenGL Learning Playground Project

This is my (Xilerast's) playground for learning OpenGL. I've been following [this tutorial](https://www.youtube.com/playlist?list=PLn3eTxaOtL2PHxN8EHf-ktAcN-sGETKfw) for the most part, while adding to the code from my own research on the internet.

It's recommended to build this project using LLVM/Clang.

## Dependencies

[GLFW](https://www.glfw.org/) \
[GLAD](https://glad.dav1d.de/) \
[stb_image](https://github.com/nothings/stb)

### Images

You will need to store on the root of the project a folder called `img`. That folder will contain the images that this project will use. (WEBP files probably not allowed)

Image names: 

"kitty.png" \
"mask.jpg"
